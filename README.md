# RSI Divergence Indicator

اندیکاتور واگرایی RSI برای TradingView (نوشته‌شده با Pine Script v5).

واگرایی بین **قیمت** و اندیکاتور **RSI** را تشخیص می‌دهد:

- 🟢 **واگرایی مثبت (Bullish)** — قیمت کفِ پایین‌تر می‌سازد ولی RSI کفِ بالاتر → احتمال برگشت صعودی.
- 🔴 **واگرایی منفی (Bearish)** — قیمت سقفِ بالاتر می‌سازد ولی RSI سقفِ پایین‌تر → احتمال برگشت نزولی.

## نصب

1. در TradingView منوی **Pine Editor** را باز کنید.
2. محتوای فایل [`RSI_Divergence.pine`](RSI_Divergence.pine) را کپی و پیست کنید.
3. روی **Add to chart** بزنید.

## تنظیمات

| ورودی | پیش‌فرض | توضیح |
|---|---|---|
| دوره RSI | 14 | طول محاسبه RSI |
| پیوت چپ/راست | 5 / 5 | حساسیت تشخیص سقف و کف |
| حداقل/حداکثر فاصله | 5 / 60 | فاصله مجاز بین دو پیوت |
| نمایش مثبت/منفی | روشن | خاموش/روشن کردن هر نوع واگرایی |

## هشدارها (Alerts)

دو شرط هشدار داخل اندیکاتور تعریف شده است: «واگرایی مثبت RSI» و «واگرایی منفی RSI».
